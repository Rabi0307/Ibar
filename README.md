## Deploy in 5 Steps

### Step 1 — Get Your Free API Key
1. Go to: https://console.anthropic.com
2. Sign up → Click "API Keys" → "Create Key"
3. Copy the key (save it!)

### Step 2 — Upload to GitHub
1. Go to: https://github.com → Sign up
2. Click "New" → Name it: ibar → Create
3. Upload all files from this folder

### Step 3 — Deploy on Vercel
1. Go to: https://vercel.com → Sign up with GitHub
2. "Add New Project" → Select "ibar" → "Import"

### Step 4 — Add API Key (IMPORTANT!)
Before clicking Deploy:
- Scroll to "Environment Variables"
- Name:  ANTHROPIC_API_KEY
- Value: paste your key from Step 1
- Click Deploy ✅

### Step 5 — Open on Your Phone
- Vercel gives you: https://ibar-xyz.vercel.app
- Open in phone browser
- Tap Share → "Add to Home Screen" 📱
